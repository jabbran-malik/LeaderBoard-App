# Leaderboard App

A simple **Leaderboard** web app built with:
- JavaScript (ES6 Modules)
- Webpack
- LocalStorage for saving scores

## Features
- Add player name and score
- Refresh list dynamically
- Input validation for name and score

## Setup
1. Clone the repo
2. Run `npm install`
3. Run `npm start` to start dev server
